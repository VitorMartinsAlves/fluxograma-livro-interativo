# Fluxograma do Livro Interativo

Página em React + TypeScript que apresenta o fluxo funcional e técnico do projeto Livro do Aluno Interativo.

## Conteúdo

- fluxo do administrador;
- fluxo do aluno;
- acompanhamento de professor e coordenação;
- tecnologias recomendadas;
- pipeline de processamento de PDF e áudio;
- estrutura das principais tabelas;
- decisões técnicas e cronograma.

## Tecnologias

- React;
- TypeScript;
- Vite;
- React Router;
- CSS responsivo.

## Executar localmente

```bash
npm install
npm run dev
```

## Gerar build

```bash
npm run build
npm run preview
```
